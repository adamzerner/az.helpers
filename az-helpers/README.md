# AZ Helpers

## Modules

### Collapsible
To use:
```
<ol az-collapsible>
  <li>one</li>
  <li>two</li>
  <li>three</li>
</ol>
```

The +/- has a class of `az-collapsible` on it. You'll probably want to use that class to position and style the element.

### Debug
To use:
```
<az-debug name="users" content="vm.users"></az-debug>
```
Or if you want a `<pre>` instead of a `<button>`:
```
<az-debug name=""
```
